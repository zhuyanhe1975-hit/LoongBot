# 🐉 LoongBot  
工业机械臂正向设计与智能控制生态系统

> 任务驱动正向设计 · 可用于控制的数字孪生 · 工业级智能控制

## LoongBot 是什么？

LoongBot 是一个面向工业机械臂的正向设计与智能控制生态系统，  
从任务需求出发，构建 **设计 → 建模 → 控制 → 实机运行** 的完整闭环。

不同于以控制算法为中心的传统框架，LoongBot 关注：
**机器人如何被设计、如何被建模，以及如何安全地运行在真实工业环境中。**

---

## 系统模块

| 模块 | 定位 | 功能描述 | 状态 |
|---|---|---|---|
| **Loong-Box** | 实时控制盒子 | 负责机械臂实时运动与硬件控制 | 待发布 |
| **Loong-Net** | 智能网络控制 | 用学习方法补偿动力学误差 | 开发中 |
| **Loong-Eye** | 数据安全模块 | 监控运行状态并触发安全保护 | 开发中 |
| **Loong-Twin** | 数字孪生模型 | 提供接近实机的仿真模型 | 开发中 |
| **Loong-Studio** | 正向设计平台 | 用于算法设计、仿真和部署 | 开发中 |

---
推荐模块排序（严格按：设计 → 仿真 → 控制 → 实机 → 监控）
1️⃣ Loong-Studio —— 设计

正向设计入口

定义任务、算法结构、参数配置

生成设计与配置结果
👉 一切从这里开始

2️⃣ Loong-Twin —— 仿真

数字孪生与验证

构建接近实机的仿真模型

验证控制与参数是否可行
👉 设计结果在这里被“试错”

3️⃣ Loong-Net —— 控制

智能控制与模型补偿

提供学习增强的动力学模型

参与控制计算或控制补偿
👉 决定“怎么控得更准”

4️⃣ Loong-Box —— 实机

实时执行与硬件控制

实时控制循环

驱动真实机械臂运行
👉 唯一直接碰硬件的模块

5️⃣ Loong-Eye —— 监控

运行期监控与安全

监控状态、检测异常

触发保护与回退
👉 全程伴随，但逻辑上最后一层
